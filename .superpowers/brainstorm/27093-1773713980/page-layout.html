선거회차 필터 동작

기본 상태

• "선거회차" Chip 표시 (기존 ui/chip.tsx 재사용)
• 초기값: 전체 (모든 회차 표시)
• ChevronDown 아이콘 포함

클릭 시

• 드롭다운 열림 (Popover 패턴)
• 회차 목록: "전체", "제 20대", "제 19대", "제 18대" ...
• 선택 시 → Chip에 선택값 표시 + 목록 필터링
• Chip이 active 상태로 전환

탭 동작

• 기존 ui/tabs.tsx의 line variant 사용
• "후보자 목록" 탭 → CandidateGrid 렌더링
• "통계분석" 탭 → "준비 중입니다" placeholder 텍스트
• 탭 전환은 Radix Tabs가 자체 관리 (controlled 모드 불필요)
